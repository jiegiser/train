# 项目说明

基于 Spring boot 3.x + SCA 2023 构建的 Spring Cloud 微服务架构项目

## 版本依赖
SCA 版本 [2023.0.3.4](https://github.com/alibaba/spring-cloud-alibaba/releases/tag/2023.0.3.4)，对应的组件版本以及 Spring boot GA 版本如下：

| **SCA** | **Spring Cloud** | **Spring Boot** | **Sentinel** | **Nacos** | **RocketMQ** | **Seata** |
| :---: | :---: | :---: | :---: | :---: | :---: | :---: |
| 2023.0.3.4 | 2023.0.3 | 3.5.9（3.2.9）| 1.8.6 | 2.4.3 | 5.1.4 | 2.1.0 |
